プログラム復習コンテンツ

setup()関数

サンプルコードの結果

サンプルコード

　　 int x=240;

　　 int y=60;

　　 int z=100;

　

　　 void setup(){

　　 　size(480, 120);

　　 　fill(102);

　　 　}

　　 void draw(){

　　 　background(204);

　　 　ellipse(x, y, z, z);

　　 　}

説明文

　draw()関数と異なり1回のみ動作する関数。

　一般的にsetup()関数内のコードはsize関数やbackground関数といった色を定義する場合が多い。

　変数はsetup関数とdraw関数の外側に定義することが多い。　

　processingが実行する順番は以下の通りになる。

　①setup()とdraw()の外側に宣言された変数を作成

　②setup()内のコードを１回実行　

　③draw()内のコードを繰り返し実行
